# Flutter Shopping App 🛍️

A modern Flutter shopping application built with Flutter.

## Features

- Splash Screen
- Onboarding Screen
- User Signup
- User Login
- SharedPreferences Authentication
- Password Hide/Show
- Form Validation
- Responsive UI

## Tech Stack

- Flutter
- Dart
- SharedPreferences
